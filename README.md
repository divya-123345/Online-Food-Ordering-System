\# 🍔 Online Food Ordering System



\## 📌 Project Description

This is a Flask-based web application for online food ordering.  

Users can browse food items, add to cart, place orders, and track their orders.



\---



\## 🚀 Features

\- User Registration and Login

\- Food Menu Display

\- Add to Cart System

\- Order Placement

\- Order Tracking

\- Admin Dashboard



\---



\## 🛠️ Technologies Used

\- Python (Flask)

\- MySQL Database

\- HTML, CSS, JavaScript

\- Bootstrap
